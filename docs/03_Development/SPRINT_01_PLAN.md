# Nexora HRMS
# Sprint 1 Plan

Sprint Name:
Authentication & Organization Setup

Status:
In Progress

---

## Sprint Goal

Allow a new organization to register, create its first administrator account, and securely access its own workspace.

---

## Features

### Authentication

- User Registration
- User Login
- User Logout
- Forgot Password
- Email Verification
- Protected Routes

---

### Organization Setup

- Create Organization
- Organization Profile
- Company Branding
- Default Settings

---

### User Profile

- Create Profile
- Link Profile to Authentication
- Link Profile to Organization

---

### Dashboard

- Welcome Screen
- Organization Name
- Logged-in User

---

## Database Tables

- organizations
- organization_settings
- profiles
- roles
- permissions
- role_permissions
- user_roles

---

## Success Criteria

A company should be able to:

- Register
- Verify Email
- Login
- Create Organization
- Access Dashboard
- See only its own data

---

## Risks

- Incorrect organization linkage
- Missing Row Level Security
- Authentication failures

---

## Definition of Done

Sprint 1 is complete when a new organization can create an account and securely access its own Nexora workspace.
